功能特性

MaoYun 是基于 sing-box-lib 开发的网络代理工具，支持用户自定义配置并一键启动 VPN 服务。

支持协议： SS, TUIC, VMess, VLESS, Trojan, Hysteria, Hysteria2

配置格式： JSON

{
  "log": {}, "dns": {}, "ntp": {}, "certificate": {},
  "endpoints": [], "inbounds": [], "outbounds": [],
  "route": {}, "services": [], "experimental": {}
}

扩展支持：

  • DNS 设置、路由规则、Fake IP 功能
  • Clash API 支持 Yacd 面板访问
  • V2Ray API 支持统计、状态查询

隐私政策

本地配置存储：用户导入或生成的配置文件将保存在本地设备，不会上传或分享。

本地日志记录：运行中产生的日志也仅保存在本地，用户可自行查看与删除。

不收集用户隐私：不采集 VPN 使用过程中的任何数据，包括访问内容、IP、连接记录等。

用户责任：请确保您的代理配置来源合法，使用行为符合所在地法律法规。
